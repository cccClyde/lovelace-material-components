## 📦 Installazione

### ☁️ Metodo 1: HACS (Consigliato)

Il modo più semplice e veloce per iniziare. Nessun download richiesto!

[![My Home Assistant](https://my.home-assistant.io/badges/hacs_repository.svg)](https://my.home-assistant.io/redirect/hacs_repository/?repository=lovelace-google-components&owner=giovannilamarmora&category=Plugin)

---

### ☁️ Metodo 2: CDN

Il secondo modo più semplice e veloce. Anche qui nessun download richiesto!

1. **Apri Home Assistant** e vai su:

   ```
   Impostazioni → Dashboard → Gestisci Risorse
   ```

2. **Clicca su "Aggiungi Risorsa"** e inserisci:
   - **Tipo:** `JavaScript module`
   - **URL:**
     ```
     https://cdn.jsdelivr.net/gh/giovannilamarmora/lovelace-material-components@master/dist/material-components.js?v=20250626
     ```

3. **Salva** e riavvia Home Assistant.

> 💡 **Suggerimento:** Modifica il parametro `?v=YYYYMMDD` per forzare l'aggiornamento della cache dopo nuove release.

---

### 🔧 Metodo 2: Installazione Manuale

Per chi preferisce ospitare i file localmente.

1. **Scarica** il file [`material-components.js`](https://github.com/giovannilamarmora/lovelace-material-components/blob/master/dist/material-components.js)

2. **Copia** il file nella cartella `config/www/` di Home Assistant:

   ```
   config/www/material-components.js
   ```

3. **Registra la risorsa** andando su:

   ```
   Impostazioni → Dashboard → Gestisci Risorse
   ```

4. **Aggiungi Risorsa**:
   - **Tipo:** `JavaScript module`
   - **URL:** `/local/material-components.js`

5. **Salva** e riavvia Home Assistant.
